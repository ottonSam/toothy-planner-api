# Specs

Esta pasta centraliza as specs do projeto.

As specs devem ser numeradas sequencialmente no formato:

```text
001-nome-da-funcionalidade.md
002-outra-funcionalidade.md
```

Antes de criar uma spec, o descritivo da funcionalidade deve estar completo. Se
alguma informacao estiver ausente, ambigua ou pouco clara, pergunte antes de
criar o arquivo.
